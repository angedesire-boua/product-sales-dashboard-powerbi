# 📊 Dashboard d’analyse des ventes de produits (Power BI)

## 📌 Vue d’ensemble du projet

Ce projet présente un **dashboard interactif Power BI** conçu pour analyser les performances commerciales liées aux ventes de produits.

Le tableau de bord fournit des **indicateurs clés** sur :

- le chiffre d’affaires
- la demande produit
- les performances par catégorie
- l’évolution des ventes dans le temps

L’objectif principal de ce projet est d’aider les décideurs à **suivre la performance commerciale** à travers des visualisations claires, dynamiques et interactives.

En analysant les données transactionnelles, ce dashboard met en évidence des indicateurs métier importants tels que :

- le **chiffre d’affaires total**
- le **nombre de commandes**
- la **quantité totale vendue**
- les **produits les plus performants**
- la **contribution des catégories au chiffre d’affaires global**

---

## 🎯 Objectifs du projet

Les principaux objectifs de ce projet sont :

- suivre la **performance globale des ventes**
- identifier les **produits les plus vendus**
- analyser la **répartition des ventes par catégorie**
- suivre l’**évolution des ventes dans le temps**
- proposer des **filtres interactifs** pour une meilleure exploration
- faciliter la **prise de décision basée sur les données**

## 🛠️ Outils et technologies utilisés

Les outils et technologies suivants ont été utilisés dans ce projet :

- **Power BI** – visualisation de données et création du dashboard
- **Fichiers CSV / Excel** – source des données transactionnelles

- **Techniques de Business Intelligence** – suivi des KPI et analyse visuelle

---

## 📂 Description du jeu de données

Le jeu de données utilisé dans ce projet contient des informations transactionnelles sur les ventes, notamment :

- les informations relatives aux **clients**
- les détails des **produits**
- les **quantités commandées**
- le **chiffre d’affaires généré**
- la **catégorie de produit**
- le **mode de livraison**
- la **date de transaction**

Ces données permettent de réaliser une **analyse complète des ventes** et d’identifier les tendances de consommation.

---

## 📈 Indicateurs clés de performance (KPI)

Le dashboard met en avant plusieurs métriques métier importantes :

- **Chiffre d’affaires total** – valeur globale des ventes réalisées
- **Nombre de commandes** – nombre total de transactions effectuées
- **Quantité totale vendue** – volume global de produits vendus

Ces KPI offrent une **vue rapide et synthétique** de la performance commerciale de l’entreprise.

---

## 📊 Fonctionnalités du dashboard

Le dashboard Power BI contient plusieurs composantes analytiques importantes :

### 1. 🔝 Top 5 des produits les plus vendus
Permet d’identifier les produits générant le plus de chiffre d’affaires.

### 2. 📦 Ventes par catégorie
Analyse les ventes totales générées par chaque catégorie de produits.

### 3. 🥧 Répartition des catégories
Montre la contribution de chaque catégorie au chiffre d’affaires global.

### 4. 📅 Évolution mensuelle des ventes
Suit les ventes dans le temps afin d’identifier les tendances ou effets saisonniers.

### 5. 🚚 Filtre par mode de livraison
Permet à l’utilisateur d’explorer les données selon le mode d’expédition utilisé.
